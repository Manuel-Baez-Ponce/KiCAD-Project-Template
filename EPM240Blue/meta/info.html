Altera EPM240 V1.4 Blue Cape

4x20-pin Expansion Board

This project template forms the basis of an expansion board (MASK) for the EPM240_V1.4 Blue board CPLD with four 20-pin GPIO header.
Note: for the EPM240 mini board, see the Altera EPM240 - Red.


This project includes a PCB edge set according to the edge board, with 4 connectors placed correctly to align the two boards, an alternative edge set is included in the User.Drawings layer for less the JTAG connector and/or the switch accessibles.
