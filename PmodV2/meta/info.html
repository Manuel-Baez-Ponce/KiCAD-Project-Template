Digilent Basys2 Dual version Pmod

1x6-pin Expansion Board

This project template forms the basis of an expansion board (Pmob) for the Digilet Basys2 an FPGA evaluation board with 4 6-pin connectors.
Note: As dual version this desing was thinking to use only 2 of 4 connectors for desings with a diferent number of connectors, see the single or full versions.


This project includes a PCB edge (recomended) with two 6-pin connectors placed correctly to align the two boards.
